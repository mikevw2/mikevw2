### Selected work

**Plant Operations Hub** — *connected-worker platform for aseptic dairy manufacturing*
Scoped four operational gaps with the stakeholders who owned them (training, sanitation, QMS) and shipped a unified platform: role-based training and certification with multi-party sign-off, a digital master sanitation schedule across six departments, and filler-line paperwork with **in-spec gating that hard-blocks out-of-spec production**. Role-based access, offline-first, bilingual (EN/ES), single self-contained file with zero install footprint. *Private repo — available for a live walkthrough on request.*

**ProcedureIQ** — *AI SOP generator for food & dairy manufacturing*
Hybrid AI + template engine generating audit-ready SOPs aligned to SQF, FSMA, PMO, and FDA 21 CFR 117. Deterministic templates handle structure; the Claude API handles narrative — driving token cost to near-zero for standard documents. Renders a full 10-section CIP SOP in ~2 ms.

**BookkeeperOS** — *AI-powered practice-management SaaS*
Multi-tenant platform for solo bookkeepers and small firms — 16 Express route modules, 22 Prisma models with full relational integrity, a 17-page React 19 front end, Stripe billing, and GPT transaction categorization with a per-user learning loop.

**SafeOps** — *offline-first food-safety & operations suite*
A ~30-app platform for aseptic dairy manufacturing spanning the production run lifecycle, inventory, food safety/QA, maintenance, and safety/HR. Each app a single self-contained file — no server, no framework, no runtime build — with cross-app integration and a platform-wide auto-generating corrective-action register.

**Almsly** — *anonymous mutual-aid desktop app*
Electron + Supabase app for a UK charity enabling anonymous mutual-aid requests funded by aggregated micro-donations, with anonymity enforced at the database layer via Row-Level Security.
